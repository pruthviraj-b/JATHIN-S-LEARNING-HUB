# Home Tuition Management (Starter)

This repository contains a starter monorepo for a Home Tuition Management system.

Structure:
- `backend/` — Node.js + Express + Prisma API
- `frontend/` — Next.js frontend

This seed includes core Prisma models, minimal auth routes, and basic frontend pages for login and dashboards.

See `backend/README.md` and `frontend/README.md` for setup steps.
